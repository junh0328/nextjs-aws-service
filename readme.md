## 👉🏼 주요 기술 & 라이브러리

### 프론트엔드 부분

- Next.js (Create-next-app)
- antd (CSS 프레임워크)
- styled-componenets (CSS 스타일 관리)
- axios (서버간 데이터 송수신)
- immer (데이터 불변성 관리)
- moment (날짜 관리)
- react-slick (이미지 캐루셀)
- swr (상태 관리)
- redux (비동기 처리)
- redux-saga (비동기 처리 +)
- redux-devtools-extension (크롬 브라우저 외부 서비스 사용하여 상태 관리, 추적)

### 백엔드 부분

- node.js
- nodemon
- express
- express-session
- sequelize
- mysql2
- multer
- cors
- cookie-parser
- passport

### aws 부분
